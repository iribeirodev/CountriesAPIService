# CountriesAPIService
Java Spring CRUD REST API

# Endpoints:

- GET http://localhost:8080/cities
- GET http://localhost:8080/cities/1
- POST http://localhost:8080/cities/1
  {
    "name": "Chicago",
    "population": 2700000
  }
- PUT http://localhost:8080/cities/1
  {
    "name": "Chicago",
    "population": 3000000
  }
  
- DELETE http://localhost:8080/cities/1
